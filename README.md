PageRank is a type of algorithm that estimates the importance of web pages based
on their links and relevance with other connections. 

Stochastic and Distribution are PageRank algorithms.
***
### Stochastic PageRank Algorithm
#### **PageRank estimation on random walkers**

Paste into terminal:

=== python page_rank.py school_web.txt ===

(Default: repeats = 1,000,000, number = 20)

***
### Distribution PageRank Algorithm
#### **PageRank estimation through probability distribution**

Paste into terminal:

=== python page_rank.py school_web.txt ===
